Kestra currently supports the following programming languages: Python, R, Node.js, Shell, Powershell, Julia and Ruby.

1. [Python](https://kestra.io/plugins/plugin-script-python)
2. [R](https://kestra.io/plugins/plugin-script-r)
3. [Node.js](https://kestra.io/plugins/plugin-script-node)
4. [Shell](https://kestra.io/plugins/plugin-script-shell)
5. [Powershell](https://kestra.io/plugins/plugin-script-powershell)
6. [Julia](https://kestra.io/plugins/plugin-script-julia)
7. [Ruby](https://kestra.io/plugins/plugin-script-ruby)
