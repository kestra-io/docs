Manage Logs generated by tasks.

On the **Logs** page, you have access to all task logs. Investigate logs with ad-hoc or saved filters, search for known IDs, and refresh data manually or set automatically periodically refresh.

On this page, you can filter by:
- Namespace
- Log level
- Interval
- Scope
- Trigger ID
- Flow ID

You can filter the displayed logs with comma-separated OR-conditions and spaced-separated AND-conditions. The following demo shows the filters in action:
