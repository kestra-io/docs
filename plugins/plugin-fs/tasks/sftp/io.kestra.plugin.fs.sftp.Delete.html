    Delete 

    type: "io.kestra.plugin.fs.sftp.Delete"

    Delete a file to a SFTP server.

    # Examples

    id: "delete"
type: "io.kestra.plugin.fs.sftp.Delete"
host: localhost
port: 22
username: foo
password: pass
uri: "/upload/dir1/file.txt"
