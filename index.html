The
orchestrator & scheduler platform.

Kestra is an infinitely scalable orchestrator & scheduler platform, in order to create, run, schedule and monitor millions of complex pipelines.

Get Started Discover Demo
Any kind of workflow

Workflow can start simple to more complex with branching, parallel, dynamic tasks, depending flows, ...

Easy to learn

Flow are simple descriptive language defined in YAML, we don't need to be a developer to create a new flow.

Easy to extend

Plugins are class first in Kestra, many are available from Kestra core team but you can create one easily.

Any triggers

Kestra is event based at heart, you can trigger an execution from api, schedule, detection, events, ...

A rich user interface

The built-in web interface allow you to create, run & monitor all your flows, no need to deploy your flow, just edit them.

Enjoy infinitely scalability

Kestra is built around top cloud native technologies, scale to million of executions without the pain.
