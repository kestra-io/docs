# Variables

The following table lists the configurable Variables of the Kestra.io and their default values.

| Parameter                              | Description                                                                                                                  | Default                                           |
| -------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------- |
| `fullnameOverride`                     | Override the full resource names                                                                                             | `{release-name}-traefik` (or traefik if release-name is traefik) |
| `image`                                | Traefik image name                                                                                                           | `traefik`                                         |
| `imageTag`                             | The version of the official Traefik image to use                                                                             |                                                   |
