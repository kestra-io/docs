Here’s what’s new:

🧑‍🔬 **Playground (Beta):** Build flows iteratively, one task at a time.
💾 **Task Caching:** Reuse results of heavy tasks with a single flag.
🔽 **Dynamic dropdowns:** Populate dropdowns with real-time data from external APIs.
🛠️ **Official SDKs:** Build on Kestra’s API with new SDKs for Java, Python, JavaScript, and Go.
🔌 **Kestra plugin:** Interact with flows and namespaces within tasks.
💬 **Improved Slack Integration:** Send rich Slack updates with execution results.
📤 **CSV export:** Download dashboard tables in one click.
📂 **Universal file protocol:** Consistent access to local and namespace files
🔎 **Execution dependency view:** Trace execution dependencies visually at a glance.
🧩 **New & upgraded plugins:** VMware (EE), Notion, Mistral, Anthropic, Perplexity, Deepseek, Databricks CLI, plus scripting for PHP, Perl, Lua, Bun and more.
🏢 **Apps Catalog (EE):** Share and discover Apps company-wide.
🔗 **Custom UI Links (EE):** Add helpful links to the sidebar.
🧪 **Unit Test Enhancements (EE):** Assert on outputs, view previous test runs.
🔒 **Mandatory Authentication (OSS):** Secure login required by default.
