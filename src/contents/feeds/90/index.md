Here’s what’s new:

🔍 **Redesigned Filters** with simple drop-downs, saved filters, and column customization.
📊 **Custom Dashboards** can now be created from the No-Code editor without writing YAML.
🎨 **Apps Catalog (EE)** now allows you to set custom colors, a logo, and app thumbnails to align the UI with your organization's brand.
🧍 **Human Task (EE)** ensures that paused executions can be manually approved only by authorized users or groups.
🧠 **AI Agents** can now **use other agents** or **A2A clients as tools** for complex AI workflows.
🤖 **Fix with AI** offers AI-powered suggestions when tasks fail.
📧 **MailReceivedTrigger** can start workflow executions in response to incoming emails.
📂 **Enhanced File Detection** can react to both new and updated files across all storage systems.
🔒 The UI and blueprints have been optimized for **airgapped environments**.
🧩 **New Plugins** – 30+ new integrations across databases, APIs, and AI: Liquibase, dlt, Stripe, YouTube, Odoo, Meta, Airtable, Flink, Cloudflare Workers AI, Oracle GenAI, LocalAI, and more.
