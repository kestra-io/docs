What's New in Kestra 0.11.0

🔹**Redesigned Topology View**
Enhanced visual navigation for data pipelines.
Expand and collapse task groups, triggers, conditional branches, and more.

🔹** Outputs Preview**
View API payloads, tables, plots in ION, CSV, JSON, PNG, JPG, text, markdown formats.
Accessible through the Execution page.

🔹**New Administration UI**
Easier troubleshooting of event-driven workflows.

🔹**VS Code Extension**
Autocompletion and syntax validation.

🔹** CLI Enhancements**
New tasks for AWS, Google Cloud, Microsoft Azure, dbt, and Spark.

🔹**Data Lake Management**
Databricks and AWS Athena integrations.

🔹**Extended Language Support**
Script Plugin now also supports Julia.

🔹**New Plugins and Tasks**
Hightouch Plugin, Telegram Notification Task, Singer Taps for Oracle & Microsoft SQL Server, GCP Tasks for PaLM 2 Large Language Models.

🔹**UI Updates**
Customizable environment name and color in the UI for better environment separation.
