**From this release on, Kestra runs on Java 17.**

**notable enhancements include:**

- Flow Metrics tab for better tracking and visualization of key metrics.
- The ability to save your flows as drafts, even if there are errors.
- UX improvements, a.o. a resizable editor window, and the ability to re-execute flows with previous inputs.
- New plugins including Git, Couchbase, and NATS 
- Improved documentation, and a new UI guide.

**For the Enterprise Edition (EE) users:**
- New License System
- Improved Inherited Variables Accessibility
- Dedicated Namespace Dashboard
