Here’s what’s new:

🧪 **Unit Tests for Flows (Beta):** Declaratively define unit tests for your flows with fixtures and assertions; run them directly from the UI or API.
🖥️ **Multi-Panel Editor:** Open, reorder, and manage multiple panels (Code, No-Code, Files, Docs, and more) side by side.
📝 **No-Code Forms:** Build and edit flows using a redesigned, form-based UI. No YAML required.
🔒 **Tenant-Based Storage Isolation:** Store workflow inputs/outputs in isolated storage for strict data separation.
🔍 **New UI Filters:** Faster autocompletion and now editable as plain text for easier configuration.
📊 **Customizable Dashboards:** Set a default dashboard, add new KPI charts, and adjust chart widths.
🐍 **Python Dependency Caching:** Automatic caching of script dependencies for faster Python task execution.
🔗 **Git Sync for Dashboards & Apps:** Version control your dashboards and apps using Git tasks.
🧩 **New & Improved Plugins:** Salesforce, HubSpot, Ollama, OpenAI, LangChain4j, GitHub Actions, Jenkins, Go scripts, InfluxDB, GraphQL, Databricks, Redis, ServiceNow, and more.
📈 **Additional enhancements:** Pebble function autocompletion, Plugin usage metrics,  detailed worker info, improved data backup/restore, and more.

Huge thanks to the open-source community and all contributors!
