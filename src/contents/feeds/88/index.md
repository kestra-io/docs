Here’s what’s new:

🤖 **AI Copilot**: Generate and edit flows from a natural language prompt.
🧠 **AI Agents**: Build custom AI agents with access to tools, memory, and LLMs.
🔎 **AI-powered docs search**: Search Kestra docs via natural language prompts.
🔗 **MCP Server**: Connect to AI IDEs & frameworks.
⚙️ **Dynamic input defaults:** Dynamically set defaults via Pebble expressions.
🗺️ **Redesigned dependency view:** More scalable flow dependency graphs with improved navigation.
✅ **Now GA:** Playground, Flow SLAs, Plugin Versioning (EE), Unit Tests (EE).
☸️ **New Helm charts**: one for development (Postgres & MinIO), one for production.
🔁 **Git sync:** Sync flows, apps, tests, and dashboards with new TenantSync and NamespaceSync tasks in the Git plugin.
🧩 **New plugins & tasks**: GitLab, Supabase, Trivy CLI & core tasks to RunTests, Kill executions and PurgeKV pairs.
🐳 **Multiple SemVer tags:** Use v0.24 or v1.0 to always pull the latest patch (e.g., 0.24.4 or 1.0.0) without pinning the exact patch version.
✨ **UI improvements:** Multi-Panel editor for Unit Tests and Apps (*now available also in the No-Code mode*!), inherited KV pairs, namespace context in the editor, replay with edited inputs & improved topology view.
