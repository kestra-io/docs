Kestra introduces a new Weaviate plugin enabling  interaction with vector databases, critical for scaling AI applications like large language models.

 The plugin facilitates creating schemas, inserting, and querying records within a Weaviate vector database directly from Kestra, with support for various programming languages and deployment options. This integration simplifies managing vector data for AI features.
